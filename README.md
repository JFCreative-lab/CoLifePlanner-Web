# Our Love Planner Web App

This is an iPhone-installable web app. It does not need Xcode, Apple Developer, certificates, provisioning profiles, or an `.ipa`.

## What It Does

- Romantic home screen with your edited couple photo.
- Relationship date: 10-02-2026.
- Tracks days together.
- Add appointments, medication, birthdays, chores, meals, and reminders.
- Mark items done.
- Daily routines can repeat.
- Saves plans locally on the phone/browser.
- Can be added to the iPhone home screen from Safari.

## How To Put It On Your iPhone

The app needs to be hosted on an HTTPS website for iPhone home screen install.

Easy options:

- GitHub Pages
- Netlify
- Vercel

After it is online:

1. Open the website in Safari on your iPhone.
2. Tap Share.
3. Tap Add to Home Screen.
4. Name it `Our Love`.
5. Tap Add.

It will open like an app from your home screen.
